# modern-credit-bridge
Plataforma em Java 11 com Arquitetura Hexagonal e DDD para modernização de legados (Mainframe). Utiliza AWS (EKS, Lambda, SQS) e persistência poliglota (RDS, DynamoDB, Neptune) para alta performance e escalabilidade. Foco em Event-Driven, CI/CD via GitHub Actions e observabilidade avançada com Datadog para garantir resiliência e baixa latência.
